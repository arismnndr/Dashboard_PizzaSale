Pizza Sales Dashboard (Proyek Studi Independen)
Dashboard analitik interaktif yang dibuat untuk memvisualisasikan data penjualan pizza, menampilkan KPI utama, tren penjualan, dan pemfilteran data dinamis. Proyek ini dikembangkan oleh Team 22 - Section Palembang sebagai bagian dari program MSIB Studi Independen RevoU Tech Academy.
